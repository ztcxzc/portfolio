# Portfolio Showcase

A preview-first portfolio website for presenting **website projects, mobile apps, and Mac apps** in one polished gallery.

## What it includes

- A bold hero section with a floating navigation bar
- Filterable project gallery for websites, mobile apps, and Mac apps
- Featured project stage with larger preview and supporting details
- Visual project cards that use platform-specific preview frames
- Responsive layout with reduced-motion support

## Tech stack

- Electron
- React 18
- webpack 5

## Getting started

```bash
npm install
npm run build
```

To launch it inside the Electron shell:

```bash
npm start
```

## Key files

- `src/App.jsx` - portfolio content, filtering, and preview layout
- `src/styles/main.css` - full visual system and responsive styling
- `public/index.html` - metadata, font loading, and app title
